This makes up my starting code for 7DRL 2026. I threw a couple of these
together last week, and some of it is is ripped from other projects I have.
On the whole, it does less than libtcod, so I don't feel bad about using it.

I have some 8x8 2-bit graphics I plan on reusing too, but they aren't included
here. I made a lot of them last summer for another jam, went back and added
more, and never used them. They should be fine for this, given I can't stand
ASCII.

Everything here is licensed under a zero-clause BSD license, and is also
largely untested. Use at your own risk!
